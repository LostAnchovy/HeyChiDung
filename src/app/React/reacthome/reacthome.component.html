Types of React Components

As we covered, components are modularized blocks of code that make up your entire view. However, each component is made up of four files: Css, Html, Spect.ts, and a TS (Typescript) files. Each of the file plays and important role, however, for now lets just focus on the three main ones, Css, Html, and TS.

The HTML is the basic mark up language, Css is the presentation layer, and the TS files is where your application functions will reside. Functions such as Logout, API request to your server, and connecting to services that you may want to build. We will dive into that later.
