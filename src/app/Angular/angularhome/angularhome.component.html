Angular Defined

Angular is a platform that makes it easy to build applications with the web. Angular combines declarative templates, dependency injection, end to end tooling, and integrated best practices to solve development challenges. Angular empowers developers to build applications that live on the web, mobile, or the desktop.

Angular and MVC

Based on the MVC model (Models - Views - Controllers), Angular represents the View. There are several different libraries and view engines that can be used for the view. A few examples are EJS(Embedded JavasScript), Vue.js, Handlebars, and React.Js, however, what makes Angular different is that is framework vs. a library (like React.js)

As a framework there are pre-set rules that are predetermined by the builders of the framework. In this case, google. There are benefits and negatives. The big benefit is structure but at the cost of freedom.
