Angular CLI

Getting Started

To install the Angular CLI:

npm install -g @angular/cli

Angular CLI commands

  • ng new : builds out an angular project
    • ng new project-app --routing : builds ou the project with a routing module.ts file
  • ng serve :builds an application and sets up a web server:
    • ng serve -- open : opens up window after building out application
  • ng generate : generates a new component
    • ng g c "component name" : Short cut to build angular components
  • ng build
    • ng build --watch : watches for changes in the app folder and updates the dist folder accordingly
    • ng build -- prod : production build
